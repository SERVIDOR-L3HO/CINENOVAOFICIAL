Panel de Documentación Técnica

Módulo de Películas

GET /api/movies

Obtiene el catálogo de películas populares con posters, banners y links VIP.

GET /api/movies/search?query={nombre}

Busca películas por título en la base de datos global de TMDB.

Módulo de Series

GET /api/series

Lista las series tendencia actuales con metadatos completos.

GET /api/series/{id}/episode?s={t}&e={e}

Genera el link de streaming VIP para un episodio específico (Temporada s, Episodio e).

Otros Módulos

GET /api/tv/channels

Obtiene la lista de canales de TV y redes de streaming disponibles.
